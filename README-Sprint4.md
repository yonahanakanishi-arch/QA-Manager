# QA-Manager — Sprint 4

## 追加機能

- 「＋新規登録」から開く入力モーダル
- システム、ベンダー、担当者、優先度、問い合わせ区分のマスタ選択
- 必須入力チェック
- 既存の Apps Script `create` API への登録
- 登録直後の一覧・ダッシュボード再読込

## 適用方法

このフォルダの `index.html`、`css/style.css`、`js/api.js`、`js/app.js` を既存プロジェクトの同名ファイルへ上書きしてください。

## 確認手順

1. 画面を再読込する。
2. 「＋新規登録」を押す。
3. 必須項目を入力して「登録する」を押す。
4. 受付状態・自動採番の案件が一覧へ現れること、QA履歴に「登録」が残ることを確認する。
