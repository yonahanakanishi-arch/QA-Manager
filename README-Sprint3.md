# QA-Manager — Sprint 3

## 含まれる機能

- Apps Script API の一覧・マスタ取得（既存の公開URLを維持）
- クライアント側キャッシュ
- キーワード、状態、担当者、システム、ベンダー、優先度による絞り込み
- ダッシュボードからの「回答待ち」「期限超過」絞り込み
- 未完了・回答待ち・今月完了・期限超過の集計
- 期限超過行の強調表示、件数表示、再読込

## 導入

このフォルダの内容を、既存リポジトリの同名ファイルへ上書きしてください。
`js/api.js` の `baseUrl` は既存の Apps Script 公開URLのままです。

## 動作確認

1. `index.html` をブラウザまたは Live Server で開く。
2. 一覧とダッシュボードが表示されることを確認する。
3. 各検索条件とキーワードで絞り込み、表示件数が変わることを確認する。
4. 「回答待ち」「期限超過」カードをクリックして対象案件のみ表示されることを確認する。
5. 「再読込」で一覧を再取得できることを確認する。
